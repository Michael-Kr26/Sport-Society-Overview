Hoe de urenbank wordt opgebouwd

Meegetelde uren = roosteruren + urencorrecties. Maandmutatie = meegetelde uren − (contracturen × 4,33) + directe bankcorrecties. De cumulatieve urenbank start bij de ingestelde beginstand en telt iedere maandmutatie vanaf de gekozen startmaand op. De aangeleverde medewerkers starten voorlopig op 0 uur per januari 2026; bestaande beginstanden kunnen door een admin worden ingevuld.
